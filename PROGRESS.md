# Chase Recipy progress

A mobile app that generates grocery lists from recipe videos/links, organizes meals, and makes cooking accessible to all.

- [x] Define core features (recipe input, ingredient extraction, grocery list, meal planner, accessibility).
- [x] Identify secondary features (substitutions, delivery integration, recipe sharing).
- [x] Document user personas and accessibility needs.

### 2. System Architecture
- [x] Design frontend (React Native or Flutter).
- [x] Define backend services (API Gateway, Recipe Parsing, Grocery List, Meal Planner).

## 📅 Example User Flow
1. User pastes a recipe link → app extracts ingredients.
2. App generates grocery list → grouped by store aisle.
3. User adds recipe to weekly planner → reminders set.
4. Accessibility: user can say “Cook pasta tomorrow” → app schedules it.

Make it simple to go from *“I saw this”* → *“It’s on the table.”*
